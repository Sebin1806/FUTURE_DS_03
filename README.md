# Marketing Funnel & Conversion Performance Analysis
## Project Overview

This project analyzes a marketing funnel to understand how users move from leads → deals → customers.
The goal is to identify conversion rates, drop-off points, and optimization opportunities that can help businesses improve revenue through better funnel performance.

This type of analysis is widely used in startups, SaaS companies, and digital marketing teams.

# Dataset

Marketing Funnel Dataset (Olist – Kaggle)
The dataset contains marketing-qualified leads and closed deal information.

Main tables used:

olist_marketing_qualified_leads_dataset.csv

olist_closed_deals_dataset.csv

# Tools Used

Power BI – Data modeling, DAX calculations, and dashboard creation

# Funnel Definition

The funnel stages were defined as:

Leads – Marketing Qualified Leads (MQLs)

Deals – Leads with deal records

Customers – Deals with a recorded won_date

Customers were identified as leads with a non-null won_date, indicating successful deal closure.

# Key Metrics

Total Leads

Total Deals

Total Customers

Lead → Deal Conversion Rate

Deal → Customer Conversion Rate

# Key Insights

A significant drop-off occurs between the lead and deal stages, indicating potential issues in lead qualification.

Certain business segments and lead origins show higher conversion efficiency.

Funnel performance varies across segments, highlighting areas for targeted optimization.

# Recommendations

Improve lead qualification before moving leads to sales teams.

Focus marketing efforts on high-converting segments and channels.

Optimize follow-up strategies to increase deal closures.
